# 🛰️ NU:TONIC — Interface Design Specification ## 🌍 1. Product Overview Nu:Tonic is a post-apocalyptic, geo-guessing multiplayer game where: * Earth players (“Humans”) / (“Aliens”) * Stranded orbital players (“Astronauts”) * AI opponents - NüTonic …compete to identify real-world locations from fragmented memory, satellite remnants, and distorted map data. ### Core UX Principles * Accessible & playful (low cognitive load) * Fast interactions (<2 taps to act) * Clear feedback loops * Game-first, minimal friction onboarding ## 🎨 2. Global Design System ### Color Palette (Post-apocalyptic sci-fi) * **Primary background:** #0B0F1A (deep space navy) * **Secondary panels:** #121829 (muted dark blue) * **Accent (neon cyan):** #00E5FF (interaction highlight) * **Warning/orange:** #FF7A00 (danger/sanctions theme) * **Success/green:** #00FF9C * **Error/red:** #FF3B3B * **Neutral text:** #E6EDF3 / #8A94A6 * 👉 *Use soft gradients + glow effects, avoid flat colors.* ### Typography * **Primary font:** Inter / SF Pro (readability) * **Display font:** Orbitron (for headings, sci-fi tone) * **Hierarchy:** * H1: 28–32px bold (Orbitron) * H2: 20–24px semi-bold * Body: 14–16px * Micro text: 11–12px * ⚠️ *Footgun: Avoid overusing sci-fi fonts → hurts readability. Keep Orbitron only for titles.* ### UI Elements * **Buttons:** Rounded (12–16px radius), Glow on hover/tap (neon cyan). * **Cards:** Frosted glass effect (blur + transparency), Subtle border: 1px rgba(255,255,255,0.05), Shadow: soft, diffuse. * **Inputs:** Minimal borders, Focus = neon cyan outline, Inline validation. ## 📱 3. Navigation Architecture * **Bottom Navigation Bar (persistent):** 🏠 Home, 🗺 Map, 🎮 Play, 🏆 Leaderboard, ⚙️ Settings. * ⚠️ *Footgun: Avoid deep navigation trees → max 2 levels. Keep Play button central and highlighted.* ## 🚀 4. Screen-by-Screen Specification * **4.1 Splash Screen:** Animated Earth (glitch), Logo: NU:TONIC, Subtitle: “Memory is all that remains.” * **4.2 Authentication:** Blurred Earth map background, Gamified copy (“Reconnect to Earth”). * **4.3 Role Selection:** Human, Astronaut, Alien. Cards expand on tap to show perks. * **4.4 Dashboard / Home:** Play Now (primary CTA), Daily Challenges, XP / Rank bar, Memory Stability meter. * **4.5 World Map (Core Gameplay):** Smooth zoom, Tap to place marker with ripple, Signal interference overlay. * **4.6 Game Launch Overlay:** Difficulty and Mode selection (Solo, Multiplayer, AI). * **4.7 Gameplay HUD:** Timer (top center), Hint button, Score preview, Player avatars. * **4.8 Success Overlay:** Distance result, Points earned, Flavor text (“Memory recovered”). * **4.9 Results Screen:** Ranked players, Map showing guesses with lines to real location. * **4.10 Leaderboard:** Top 3 highlighted, Titles (e.g., “Cartographer”, “Signal Ghost”). * **4.11 Settings:** Audio, Accessibility (High contrast, Reduced motion), Account. ## 🎮 5. Gamification Layer * XP + Levels, Daily missions, Unlockable themes, Cosmetic avatars. * Narrative Layer: “Sanctions have cut orbital supply chains”, “Astronauts rely on memory fragments”. ## ⚠️ 6. Common UX Footguns (Critical) 1. **Map Interaction Precision:** Enlarge touch radius invisibly. 2. **Cognitive Overload:** Keep UI readable first, thematic second. 3. **Multiplayer Sync:** Show “ghost guesses” if needed. 4. **Feedback Timing:** Always respond within 100ms. 5. **Role Confusion:** Use icons + short descriptions. ## 🏁 8. Final Design Intent * A game first, not a utility. * Lightweight but immersive. * Fast, intuitive, and replayable.
